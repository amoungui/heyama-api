# Heyama API - NestJS Backend

API REST pour la gestion d'objets avec upload d'images et WebSocket temps réel.

## 🛠️ Technologies
- NestJS
- MongoDB avec Mongoose
- MinIO (stockage S3 compatible)
- WebSocket (Socket.io)
- Multer pour l'upload de fichiers

## 🚀 Installation

```bash
# Installer les dépendances
npm install

# Démarrer en développement
npm run start:dev

# Construire pour production
npm run build

# Démarrer en production
npm run start:prod
